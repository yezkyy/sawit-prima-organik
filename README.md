# Sawit Prima Organik - Website Company Profile Premium

Website company profile premium untuk brand pupuk organik khusus kelapa sawit "Sawit Prima Organik". Dibangun menggunakan arsitektur MVC Laravel 12 yang rapi dengan fokus pada visual yang profesional dan SEO yang optimal.

## Teknologi Utama
- **Framework:** Laravel 12
- **Frontend:** Blade Templating, Tailwind CSS v3, AlpineJS
- **Database:** SQLite / MySQL
- **Assets:** Laravel Vite

## Fitur Utama
- **Public Facing Website:** Home, Produk, Galeri, Testimoni, Kontak.
- **Custom Admin Panel:** Manajemen Banner, Galeri, Testimoni, dan Pengaturan Website.
- **SEO Optimized:** Meta tags dinamis, Schema.org Organization, Sitemap.xml.
- **Premium Design:** UI modern bertema agrikultur profesional.
- **WhatsApp Integration:** Tombol CTA mengambang dan form kontak.

## Instalasi
1. Clone repository.
2. Jalankan `composer install` & `npm install`.
3. Salin `.env.example` ke `.env` dan sesuaikan database.
4. Jalankan `php artisan key:generate`.
5. Jalankan `php artisan migrate --seed` (Default login: `admin@sawitprima.com` / `password`).
6. Jalankan `php artisan storage:link`.
7. Jalankan `npm run dev` atau `npm run build`.
8. Jalankan `php artisan serve`.
